Detailed Description

A class that communicates with a TcpClient object.

This class is used as a wrapper for a TcpClient object. It also represents a connected user and the client that the user is using. The object contains the following global variables:

  • innerClient: the TcpClient to communicate with.
  • clientID: the ID of the client (table).
  • userID: the ID of the connected user.
